## Key

`i` 输入模式

`x` 删除当前光标所在处的字符删除

`a` 进入插入模式，在光标下一个位置开始输入文本

`o`\[`O`] 在当前行的下\[上]方插入一个新行，并进入插入模式

`dd` 剪切当前行

`yy` 复制当前行

`p`\[`P`] 粘贴剪贴板内容到光标下\[上]方

`u` 撤销上一次操作

### 光标\[可配合数字使用]

| <br />       | <br />     |
| ------------ | ---------- |
| h 或 向左箭头键(←) | 光标向左移动一个字符 |
| j 或 向下箭头键(↓) | 光标向下移动一个字符 |
| k 或 向上箭头键(↑) | 光标向上移动一个字符 |
| l 或 向右箭头键(→) | 光标向右移动一个字符 |

`n` <space><ether> n表示数字, 5, 光标会向<右><下>移动这一行的 5 个字符

`H`	光标移动到这个屏幕的最上方那一行的第一个字符

`M`	光标移动到这个屏幕的中央那一行的第一个字符

`L`	光标移动到这个屏幕的最下方那一行的第一个字符

`G`	移动到这个档案的最后一行(常用)

`nG`	n 为数字。移动到这个档案的第 n 行。例如 20G 则会移动到这个档案的第 20 行(可配合 :set nu)

`gg`	移动到这个档案的第一行，相当于 1G

`w<W>` 下一个单词

`e<E>` 词尾

`-<+>` 前<后>一行

`s<S>` 删除字符<行>并插入

`(<)>` 句首<下一句首>

### 搜索替换

`/<?>word`   像光标下<上>搜寻word的字符串

`n``N`  n下一个,N上一个

### 删除、复制与贴上

`x``X` 向后<前>删除一个字符

`nx` 向后连续删除n个字符

`<n>dd` 剪切游标所在的一<n>整行

`d1G` 删除光标到第一行的所有数据

`dG` 删除光标到最后一行的所有数据

`d$` 删除光标所在处到改行最后一个字符

`d0` 删除光标所在处到改行第一个字符

`yy` 复制光标所在的一行

`nyy` 复制光标所在的<n>整行

`y0`<$> 复制光标所在的字符到行首<尾>的所有字符

`p`<`P`> 粘贴剪贴板内容到光标下\[上]方

`J` 将光标所在行与下一行的数据合并为一行

`c` 重复删除多个数据, \[10cj]

`u` 复原前一个动作

`Ctrl+r`<.> 重做上一个动作

### 模式

`i<I>` i输入模式在当前光标处, I在所在行的第一个非空格符处开始输入

`a<A>` a从光标所在下一字符开始输入, A光标所在行最后一个字符开始输入

`o<O>` o光标所在的下<上>一行开始输入新的一行

`r<R>` r取代光标所在字符一次<多次\<ESC退出>>

### 保存

:w \[filename] 保存为另一个文件

:r \[filename] 读取另一个档案的数据

:n1, n2 \[filename] 将n1->n2的内容存储为filename这个文件

### 环境变更

:set nu<nonu> 显示<隐藏>行号
